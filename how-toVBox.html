How-to sobre virtualBox

descarga, instalacion y configuración de virtualBox

Descarga de virtualBox

Para poder descargarnos el software virtualBox, nos dirigimos a la web de VirtualBox, una vez en la web le haremos click en el apartado de Dowloads.
Una vez que estemos en el apartado de descarga procederemos a descarga la version más reciente del software de VirtualBox, haciendo click en el enlace de Windows hosts.

1. Acceder a la instalción de virtualBox

Una vez descarga el software procederemos con la instalacion, haciendo doble click al archivo, seguido se nos abririra una venta para empezar la instalacion, para empezar la instalacion le daremos click a Next

2. Configuración personalizada de la instalacion

En la siguiente ventana podemos elegir la localización de la instalacion, por de fecto ya tendra una localización preestablecida, pero podemos modifacarla a nuesto antojo

3. Interfaz de red

En la siguiente ventana se instalara la interfaz de red, en el que nos dara un aviso de que al proceder a la instalción de la función de red de Oracle VM VirtualBox, nos restablecerá nuesta conexión red y la desconectará temporalmente de la red. Le haremos click a yes para iniciar este paso

4. instalacion de dependencias de python core / win32api

En la siguiente ventana nos pedira que instalemos python core / win32api, Oracle VM virtualBox necesita para su instalacion el python bindings, y para ello requiere instalemos primero python core y win32api, daremos click a Yes para proceder con su instalacion

5. Preparado para instalar

En la siguiente ventana procederemos a la instalacion de virtualBox, ya que anterior mente hemos realizados los pasos anteriores y el software esta listo para su instalacion daremos click a Install, para ejecutar su instalacion.

6. Proceso de instalación

Al hacer click en Install se nos habrira otra ventana en la que nos pondra el proceso de instalacion y el estado en el que esta, en este parte de la instalacion no tenemos que realizar nada solo espara a que finalice la instalacion.

7. instalación finalizada

Una vez que finalice la instalacion y todo se realice con correctamente, nos saldar la ultima venta informandonos de que la instalacion se completo en esta venta tendremos una casilla en la que podemos marcarla por si queremos que al cerrar la instalacion se nos ejecute Oracle virtualBox, y por ultimo para finalizar la instalción haremos click en Finish, en el que se cerrara la ventana y si hemos marcado la casilla de Start Oracle virtualBox se nos ejecutara el software o si la dejamos desmarcada cerraremos la instalacion sin ejecutar el software.

Configuración de red-nat

En este apartado se explicara de como crear y configurar una red-nat en virtualBox. Necesitaremos configurar una red-nat para que nuestra dos maquinas virtuales se puedan comunicar entre si. Para ello, nos hiremos a herramientas en virtualBox, y clickamos en red para entrar en el menu de configuración.

Creación de red-nat

Una vez dentro del menu de red de virtualBox, clickamos en la pestaña de Redes NAT y creamos una red-nat nueva, al crearla tendremos nuestra red-nat con el nombre de NattNetwork, una vez creada solo tenemos que asignarles la red-nat a nuesto sistemas operativos Windows y Ubuntu.

Configuración de red-nat ubuntu

Para asignarle una red-nat a ubuntu, seleccionamos nuesto sistema ubuntu y hacemos click derecho y le damos a configuración, con lo que se nos abririra la ventana de configuración y nos vamos al apartado de red.

Conectar una red-nat

En la ventana de configuración de red, seleccionamos la pestaña Adaptador 1, y en la pestaña de Conectado a , en le despegable seleccionamos red-nat, con esto ya tenemos una red-nat para nuestro sistema ubuntu.

Configuración de red-nat Windows

Para asignarle una red-nat a windows, seleccionamos nuesto sistema windows y hacemos click derecho y le damos a configuración, con lo que se nos abririra la ventana de configuración y nos vamos al apartado de red.

Conectar una red-nat

En la ventana de configuración de red, seleccionamos la pestaña Adaptador 1, y en la pestaña de Conectado a , en le despegable seleccionamos red-nat, con esto ya tenemos una red-nat para nuestro sistema ubuntu.
