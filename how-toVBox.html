How-to sobre virtualBox

descarga, instalacion y configuración de virtualBox

Descarga de virtualBox

Para poder descargarnos el software virtualBox, nos dirigimos a la web de VirtualBox, una vez en la web le haremos clic en el apartado de Dowloads.
Una vez que estemos en el apartado de descarga, procederemos a descarga la versión más reciente del software de VirtualBox, haciendo clic en el enlace de Windows hosts.

1. Acceder a la instalación de virtualBox

Una vez descarga el software procederemos con la instalación, haciendo doble clic al archivo, seguido se nos abrirá una venta para empezar la instalación, para empezar la instalación le daremos clic a Next.

2. Configuración personalizada de la instalación

En la siguiente ventana podemos elegir la localización de la instalación, por defecto ya tendrá una localización preestablecida, pero podemos modificarla a nuestro antojo.

3. Interfaz de red

En la siguiente ventana se instalará la interfaz de red, en el que nos dará un aviso de que al proceder a la instalación de la función de red de Oracle VM VirtualBox, nos restablecerá nuestra conexión red y la desconectará temporalmente de la red. Le haremos clic a yes para iniciar este paso.

4. instalación de dependencias de python core / win32api

En la siguiente ventana nos pedirá que instalemos python core / win32api, Oracle VM virtualBox necesita para su instalación el python bindings, y para ello requiere instalemos primero python core y win32api, daremos clic a Yes para proceder con su instalación.

5. Preparado para instalar

En la siguiente ventana procederemos a la instalación de virtualBox, ya que anteriormente hemos realizado los pasos anteriores y el software está listo para su instalación, daremos clic a Install, para ejecutar su instalación.

6. Proceso de instalación

Al hacer clic en Install se nos abrirá otra ventana en la que nos pondrá el proceso de instalación y el estado en el que está, en este parte de la instalación no tenemos que realizar nada, solo espera a que finalice la instalación.

7. instalación finalizada

Una vez que finalice la instalación y todo se realice con correctamente, nos saldrá la última venta informándonos de que la instalación se completó en esta venta tendremos una casilla en la que podemos marcarla por si queremos que al cerrar la instalación se nos ejecute Oracle virtualBox, y por último para finalizar la instalación haremos clic en Finish, en el que se cerrara la ventana y si hemos marcado la casilla de Start Oracle virtualBox se nos ejecutará el software o si la dejamos desmarcada cerraremos la instalación sin ejecutar el software.

Configuración de red-nat

En este apartado se explicará de como crear y configurar una red-nat en virtualBox. Necesitaremos configurar una red-nat para que nuestra dos máquinas virtuales se puedan comunicar entre sí. Para ello, nos iremos a herramientas en virtualBox, y clicamos en red para entrar en el menú de configuración.

Creación de red-nat

Una vez dentro del menú de red de virtualBox, clickamos en la pestaña de Redes NAT y creamos una red-nat nueva, al crearla tendremos nuestra red-nat con el nombre de NattNetwork, una vez creada solo tenemos que asignarles la red-nat a nuestros sistemas operativos Windows y Ubuntu.

Configuración de red-nat ubuntu

Para asignarle una red-nat a ubuntu, seleccionamos nuestro sistema Ubuntu y hacemos clic derecho y le damos a configuración, con lo que se nos abrirá la ventana de configuración y nos vamos al apartado de red.

Conectar una red-nat

En la ventana de configuración de red, seleccionamos la pestaña Adaptador 1, y en la pestaña de Conectado a, en el despegable seleccionamos red-nat, con esto ya tenemos una red-nat para nuestro sistema Ubuntu.

Configuración de red-nat Windows

Para asignarle una red-nat a windows, seleccionamos nuestro sistema Windows y hacemos clic derecho y le damos a configuración, con lo que se nos abrirá la ventana de configuración y nos vamos al apartado de red.

Conectar una red-nat

En la ventana de configuración de red, seleccionamos la pestaña Adaptador 1, y en la pestaña de Conectado a, en le despegable seleccionamos red-nat, con esto ya tenemos una red-nat para nuestro sistema Windows.
