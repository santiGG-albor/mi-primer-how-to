  • configuración de windows 10

Una vez que tengamos nuestro sistema operativo instalado, nos toca configurar nuestro sistema para que pueda comunicarse con el sistema operativo ubuntu, para ello en la barra del buscador ingresamos "Firewall de windows Defender", hacemos click en ella y abriremos el panel de control del Firewall de Windows Defender.

Una vez en el panel de control de Firewall de Windows Defender, en la parte derecha nos saldra sietes opciones, las que nos interesa es la opcion de configuración avanzada, hacemos click en ella para poder entrar en el panel de configuración.

Dentro del panel de configuración avanzada, en la parte derecha en las opciones que tenemos hacemos click en el opcion de Reglas de entrada, hacemos scroll hacia habajo para buscar las reglas de entrada de "Archivos e impresoras compartidos (petición eco: ICMPv4 de entrada) y Archivos e impresoras compartidos (petición eco: ICMPv6 de entrada), y hacmos click derecho y le damos a "Habilitar regla", para Habilitar los premisos de entrada, lo hacemos en las cuatros que nos sale las dos de ICMpv4 y las dos de ICMPv6.

Una vez que le demos permisos y las cuatros reglas de compartir Archivos este habilitado se nos tiene que quedar como en el imagen que tenemos.

Una vez que tengamos los permisos habilitado, en la barra del buscador ingresamos configuración de uso compartido avanzado, buscamos el apartado compartir archivos e impresoras y activamos la opcion de "activar el uso compartido de archivos e impresoras", con todo estos paso ya nuestro sistema operativo esta listo para poder comunicarse con nuestro sistema operativo ubuntu.

Ahora vamos a comprobar que nuestro sistema operativo windows puede hacer ping al sistema operativo ubuntu, para ello abrimos Ejecutar, si no sabemos donde esta en el buscador ingresamos "Ejecutar", dentro de Ejecutar ingresamos "cmd" para que no se habrá el cmd de windows.

Dentro del cmd de windows, ingresamos ping y la ip de nuestro sistema operativo ubuntu, en nuestro caso es "ping 10.0.2.15", con esto comprobara si hambas maquinas pueden hacer ping entre ella y si nuestros sistemas operativos pueden comunicarse entre ellos.

Si todo esta correcto y hacen ping entre hambos sistemas, en Ejecutar ingresamos la ip que tiene agsinada ubuntu y el nombre de la carpeta que tenemos para compartir, en nuestro caso es "\\10.0.2.15\compartida".

Si todo esta bien nos pedira que ingresemos las credenciales de red, tendremos que poner el nombre de usuario que hemos creado en unbuntu y la contraseña que hemos creado para ese usuario.

Una vez que ingresemos la credenciales de red y este correcto se nos abrira la venta con la carpeta compartida que tenemos en ubuntu, ahora para probar que funciona correctamente, creamo un archivo de texto, si todo esta bien este archivo nos tiene que salir en la carpeta compartida que tenemos en ubuntu.

Para comprobar que el archivo de texto que hemos creado en windows esta compartido, nos vamos a nuestro sistema ubuntu, y abrimos las terminal y nos cambiamos al usuario winuser1, y nos vamos a donde tenemos nuestra carpeta compartida y hacemos un "ls" si todo esta correcto y a funcionado tendremos que ver el archivo de texto que hemos creado anterior mente.
