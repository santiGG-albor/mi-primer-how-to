  • configuración de windows 10

Una vez que tengamos nuestro sistema operativo instalado, nos toca configurar nuestro sistema para que pueda comunicarse con el sistema operativo Ubuntu, para ello en la barra del buscador ingresamos "Firewall de Windows Defender", hacemos clic en ella y abriremos el panel de control del Firewall de Windows Defender.

Una vez en el panel de control de Firewall de Windows Defender, en la parte derecha nos saldrá siete opciones, las que nos interesa es la opción de configuración avanzada, hacemos clic en ella para poder entrar en el panel de configuración.

Dentro del panel de configuración avanzada, en la parte derecha en las opciones que tenemos hacemos clic en el opción de Reglas de entrada, hacemos scroll hacia abajo para buscar las reglas de entrada de "Archivos e impresoras compartidos (petición eco: ICMPv4 de entrada) y Archivos e impresoras compartidos (petición eco: ICMPv6 de entrada)", y hacemos clic derecho y le damos a "Habilitar regla", para Habilitar los premisos de entrada, lo hacemos en las cuatro que nos sale, las dos de ICMpv4 y las dos de ICMPv6.

Una vez que le demos permisos y las cuatro reglas de compartir Archivos este habilitado se nos tiene que quedar como en la imagen que tenemos.

Una vez que tengamos los permisos habilitados, en la barra del buscador ingresamos configuración de uso compartido avanzado, buscamos el apartado compartir archivos e impresoras y activamos la opción de "activar el uso compartido de archivos e impresoras", con todo estos paso ya nuestro sistema operativo está listo para poder comunicarse con nuestro sistema operativo Ubuntu.

Ahora vamos a comprobar que nuestro sistema operativo Windows puede hacer ping al sistema operativo Ubuntu, para ello abrimos Ejecutar, si no sabemos donde está en el buscador ingresamos "Ejecutar", dentro de Ejecutar ingresamos "cmd" para que no se habrá el cmd de Windows.

Dentro del cmd de Windows, ingresamos ping y la IP de nuestro sistema operativo Ubuntu, en nuestro caso es "ping 10.0.2.15", con esto comprobará si ambas máquinas pueden hacer ping entre ella y si nuestros sistemas operativos pueden comunicarse entre ellos.

Si todo está correcto y hacen ping entre ambos sistemas, en Ejecutar ingresamos la IP que tiene asignada Ubuntu y el nombre de la carpeta que tenemos para compartir, en nuestro caso es "\\10.0.2.15\compartida".

Si todo está bien nos pedirá que ingresemos las credenciales de red, tendremos que poner el nombre de usuario que hemos creado en Unbuntu y la contraseña que hemos creado para ese usuario.

Una vez que ingresemos las credenciales de red y este correcto se nos abrirá la venta con la carpeta compartida que tenemos en Ubuntu, ahora para probar que funciona correctamente, creamos un archivo de texto, si todo está bien este archivo nos tiene que salir en la carpeta compartida que tenemos en Ubuntu.

Para comprobar que el archivo de texto que hemos creado en Windows está compartido, nos vamos a nuestro sistema Ubuntu, y abrimos la terminal y nos cambiamos al usuario winuser1, y nos vamos a donde tenemos nuestra carpeta compartida y hacemos un "ls" si todo está correcto y ha funcionado tendremos que ver el archivo de texto que hemos creado anteriormente.
