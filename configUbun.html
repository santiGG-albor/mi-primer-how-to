How-to sobre máquinas virtuales

descarga, instalacion y configuración de sistemas operativos

  • Instalación de Ubuntu

  • configuración de Ubuntu

Vamos a pemezar con la configuración de nuestro sistema operativo ubuntu, para ello abriremos nuestra terminal y ingresamos el comando su, para identificarnos con root, y poder empezar con la configuración.

Una vez que estemos como root, ingresamos el comando "apt upgrade" para poder instalar las nuevas versiones de los paquetes de los repositorios de nuestro sistemas, que necesitaremos para nuestra configuración.

Cuando ingresemos el comando llegara a una parte que nos pedira que si queremos continuar, mostrando todo lo que va instalar y actualizar, le damos a si ingresando "Y".

A continuación vamos con el comando "apt update", que nos actualizara la listas de paquete disponibles en los repositorios de ubuntu.

Una vez todo configurado nos toca instalar samba, mediante el comando "apt install samba", que es un software que implementa el protocolo SMB y permite a las distribuciones de ubuntu/linux compartir archivos e impresoras con cualquier otro dispositivo conectado a la red, que es lo que nos va a permitir que podamos cumunicar nuestros sistemas.

A continuación con todo ya instalado, vamos con la creacion del segundo usuario con el que vamos a utilizar para compartir la carpeta con nuesto sisitemas, ingresamos el comando "Adduser winuser1", con esto se no creara nuestro nuevo usuario.

Configuramos nuestro nuevo usuario, ingresando una nueva contraseña para este usuario, en el campo del nombre, telefono..., lo dejamos en blanco estos campanos no, nos interesa, para terminar nos pedira si la información es correcta y le decimos que si con "Y", con esto tenemos listo nuestro nuevo usuario para comparti la carpeta.

Ahora nos vamos a logear con nuestro nuevo usuario, mediante en el comando "su winuser1" y nos dirigimos hacia su carpeta personal con el comando "cd ..", hasta llega a la carpeta "/home", una vez en "/home" ingresamos el comando "cd winuser1" para entrar en nuestra carpeta personal.

Una vez dentro de nuestra carpeta personal vamos con la creación de nuestra carpeta que la llamaremos compartida, la creamos mediante el comando "mkdir compartida".

Una vez creada con éxito nuestra carpeta, vamos a darle permiso de lectura, escritura y ejecución para poder trabajar en ella y poder compartir los archivos, mediante el comando "chmod 777".

Ahora nos toca confugurar samba, que lo haremos mediante un archivo de texto, para ello vamos a instalar vim un editor de texto plano de linux, ingresamos el comando "Apt-get install vim", para proceder con la instalacion.

Cuando este instalado nos cambiamos a root mediante el comando "su", una vez en root vamos a editar el archivo, este archivo lo podemos editar con vim o con nano, ambos son editores de texto plano, en nuestro caso vamos a utilizar nano, ingresamos el comando "nano /etc/samba/smb.comf".

Una vez que se carge el editor de texto plano nos despalazamos hacia abajo para poder empezar con la configuración, y ingresamos lo siguiente, [compartida] path= /home/winuser1/compartida browseable = yes read only = no valid users = winuser1, una vez que lo tengamos todo solo nos toca guardar los cambiamos y salir del editor, para salir utilizamos el comando :wq.

Ahora tenemos que añadir nuestro usuario winuser1 a los usuarios de samba y ponerle una contraseña, para poder entrar en nuestra carpeta que vamos a comparti en el sistema operativo de windows, para ellos ingresamos el comando "smbpasswd -a winuser1", nos pedira que creemos una nueva contraseña, para esta contraseña podemos utilizar la misma contraseña que tenemos para nuestro usuario de winuser1.

Una vez añadido nuestro usuario en samba, nos cambiamos al usuario winuser1, dentro de nuestra carpeta compartida, creamos un archivo nuevo, para crear este archivo nuevo lo hacemos con el comando "touch archivo-linux", que este archivo sera el que veamos cuando entremos en nuestra carpeta compartida en windows.

Ya solo nos queda resetear el servicio de samba desde root, para ello nos cambiamos a root con "su", y una vez estando como root reseteamos samba con el comando "systemctl restart smbd.service".

Ahora con todo ya instalado, configurado y reseteado, nos queda mirar la dirección ip de nuesta máquina, con el comando "ip a", y la anotamos que la vamos a tener que utilizar para poder entrar en nuestra carpeta compartida desde windows.
